# MPS-Group-Project

The code used to generate the images used within the group project. These scripts use the [pygame](https://www.pygame.org/news) library for displaying images to the screen as well as for responding to user input.

# 1.3 Using self-similarity to make fractals

The figure of the [Sierpiński Carpet](https://en.wikipedia.org/wiki/Sierpi%C5%84ski_carpet) was generated by [this script](sierpinski_carpet.py) using a value of 8 for the variable named "iterations".

# 1.4 Fractals from iteration and recursion

The fractals in this section are generated using the [main fractal generating script](mandelbrot.py).

## 1.4.1 Julia and Mandelbrot sets

Figure 1.2 of the Mandelbrot Set was generated with:
```
python3 mandelbrot.py --s True --fractal mandelbrot --pixel_width 6400
```

## 1.4.2 Lyapunov fractals

Yep, they're fractals

```
python3 mandelbrot.py --s True --fractal "lyapunov" --file "lyapunov.png" --pixel_width 6
400
```
