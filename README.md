# MPS-Group-Project

The code used to generate the images used within the group project. These scripts use the [pygame](https://www.pygame.org/news) library for displaying images to the screen as well as for responding to user input.

# Section 3

The figure of the [Sierpiński Carpet](https://en.wikipedia.org/wiki/Sierpi%C5%84ski_carpet) was generated by [this script](sierpinski_carpet.py) using a value of 8 for the variable named "iterations".

## AI Usage

None of the code found in this repository was created using Artificial Intelligence of any sort, this includes writing, editing and debugging.
